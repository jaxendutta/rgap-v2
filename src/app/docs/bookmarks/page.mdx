import { LuBookmark, LuBookmarkCheck } from "react-icons/lu";

# Bookmarks

Bookmarks allow you to curate your own collections of grants and researchers.

## How it works

The bookmark button has two states. It behaves exactly like the buttons you see throughout the app.

<div className="flex flex-wrap gap-8 my-8 p-6 bg-gray-50 rounded-xl border border-gray-100 justify-center">
    <div className="flex flex-col items-center gap-3">
        <button className="flex items-center gap-2 px-4 py-2 bg-white border border-gray-200 rounded-md shadow-sm text-sm font-medium text-gray-700 hover:bg-gray-50">
            <LuBookmark className="w-4 h-4" />
            <span>Bookmark</span>
        </button>
        <span className="text-xs text-gray-500 font-mono">Default State</span>
    </div>

    <div className="flex flex-col items-center gap-3">
        <button className="flex items-center gap-2 px-4 py-2 bg-blue-600 border border-blue-600 rounded-md shadow-sm text-sm font-medium text-white hover:bg-blue-500">
            <LuBookmarkCheck className="w-4 h-4 text-blue-50" />
            <span>Bookmarked</span>
        </button>
        <span className="text-xs text-gray-500 font-mono">Active State</span>
    </div>
</div>

<Alert type="tip" title="Quick Tip">
    You can bookmark items directly from the search results page without opening the full details.
</Alert>

## Managing Your Library

Navigate to the **Bookmarks** tab in the sidebar to see everything you've saved.

<Alert type="warning">
    If you remove a bookmark that has a personal note attached to it, the note will be permanently deleted.
</Alert>
